Quiz Website

This is a dynamic quiz platform developed using HTML, CSS, and JavaScript. It allows users to take quizzes in real-time, shows instant feedback, and calculates the final score. The design is fully responsive and easily customizable.

Frontend HTML, CSS, JavaScript
Features
  • Real-time question rendering
  • Instant scoring and user feedback
  • Responsive design for both mobile and desktop
  • Easy to add or update quiz questions

This project can be used in educational platforms, training assessments, or personal learning tools. Its modular codebase makes it easy to integrate new question sets and UI customizations.
